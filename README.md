# labor-regulations

規程管理リポジトリ（labor-regulations）


■ 目的

本リポジトリは、就業規則・賃金規程・各種労使協定の変更履歴を一元管理し、
内容の正確性・透明性・追跡可能性を確保することを目的とする。


■ 管理対象

就業規則
賃金規程
各種規程
労使協定（36協定除く）
templates


■ フォルダ構成

就業規則/：就業規則
賃金規程/：賃金規程
各種規程/：国内出張旅費規程、慶弔見舞金規程など
労使協定/：各種協定書
templates/：ひな形・参考資料


■ 運用ルール（必ず守る）

ファイル名は変更しない（例：就業規則.docx）
変更時は必ずGitで履歴を残す
コミットメッセージは具体的に記載する
不要なファイルは削除前に確認する


■ 更新手順（基本フロー）

最新状態を取得
git pull
ファイルを修正（Word / Excel等）
変更を登録
git add .
履歴を記録
git commit -m "変更内容（例：第15条 勤務時間の定義修正）"
GitHubへ反映
git push


■ 推奨ルール（できれば）

重要な変更は事前に関係者へ共有
複数規程にまたがる変更はまとめて管理する
定期的に履歴を確認する


■ 注意事項

GitHub上のファイルは「最新版の共有場所」
実際の編集はローカルで行う
誤って変更した場合でも、履歴から復元可能


■ 管理責任

本リポジトリの管理はバックオフィス（人事・労務）にて行う。
変更内容については必要に応じて社内承認を経ること。
